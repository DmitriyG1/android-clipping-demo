# Android View clipping

Review of clipping methods in Android.

Examined 4 ways to implement View clipping:
* View.clipToOutline
* Canvas.clipPath()
* PorterDuff.Mode + View.LAYER_TYPE_SOFTWARE
* PorterDuff.Mode + View.LAYER_TYPE_HARDWARE
